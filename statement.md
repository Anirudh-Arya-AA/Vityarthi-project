1. Project TitleMovie Guessing Game CLI2. Problem StatementThe project addresses the need for a simple, accessible, and customizable interactive game that can be run directly from the command line. It provides a platform for users to test their movie trivia knowledge in an engaging format, while simultaneously serving as a straightforward learning exercise in Object-Oriented Programming (OOP), input validation, and algorithmic logic (obfuscation/hinting) for developers.3.
